# Cloud Tech React App

This is a React conversion of the Cloud Tech landing page, originally built with HTML and CSS.

## Features

- Fully responsive design
- Modern dark theme with blue accents
- All Font Awesome icons included
- Sections include:
  - Navigation bar
  - Hero section with laptop and phone mockups
  - About section
  - Services grid
  - Development process steps
  - Projects showcase
  - Statistics section
  - Client testimonials
  - Call-to-action
  - Footer with social links

## Technologies Used

- React 18
- Font Awesome 6.5.1
- CSS3 with Grid and Flexbox
- Responsive design with media queries

## Conversion Notes

This app is a direct conversion from the original HTML/CSS landing page (`Full.html`). All styles, icons, images, and layout have been preserved exactly as they were in the original design.
